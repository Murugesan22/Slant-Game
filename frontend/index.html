📚 Game Instructions

🎯 Game Rules

  • Fill Every Cell: Each cell needs exactly one diagonal slash (\ or /)
  • Match Numbers: Corner numbers indicate how many lines must touch that point
  • No Loops: Slashes cannot form closed circles or cycles

🎮 Controls

  • Single Click: Toggle between \ and / slash types
  • Double Click: Clear the cell
  • New Game: Start a fresh puzzle
  • Undo: Revert last move
  • Multiplayer: Play vs CPU (choose strategy)
  • Solve: Auto-complete the puzzle

💯 Scoring System

  • +1: Every valid move
  • +2: Complete a numbered constraint
  • +3: Create perfect cell (all 4 corners satisfied)
  • +1: Place near center (strategic bonus)

💡 Best moves can score up to 13 points!

🎨 Visual Feedback

  • 🟢 Green Numbers: Constraint satisfied ✅
  • 🔴 Red Numbers: Constraint violated ❌
  • 🔴 Red Slashes: Part of a loop (invalid!)

🤖 CPU Algorithms

  • Dynamic Programming: Row-by-row DP with memoization
  • Divide & Conquer: Quadrant-based recursive solving
  • Hybrid (D&C + DP): D&C partitioning with DP solving
  • MRV Backtracking: Fills most constrained cell first using Minimum Remaining Values heuristic
  • Conflict-Directed Backjumping: Jumps directly to conflict source on failure instead of chronological backtracking
  • Forward Checking: Prunes neighbour domains after each assignment; backtracks immediately on domain wipeout

💡 All solvers use pure algorithmic approaches - no greedy heuristics!
